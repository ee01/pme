**PS：老模板只有三种列表页展现形式怎么办？**((
**情况一**  如果老模板三种形式：
  * news.html比较像 图文列表（或像文字列表
  * pro.html比较像 图片列表
  * about.html是 内容单页
则可这样转换老模板：
  * [[ued:template:list|list-text.html]]       参照news.html模板
  * [[ued:template:list|list-image.html]]      参照pro.html模板
  * [[ued:template:list|list-imagetext.html]]  也参照news.html模板
  * [[ued:template:list|list-page.html]]       参照about.html模板
**情况二**  如果老模板三种形式：
  * news.html比较像 文字列表
  * pro.html比较像 图文列表（或像图片列表
  * about.html是 内容单页
则可这样转换老模板：
  * [[ued:template:list|list-text.html]]       参照news.html模板
  * [[ued:template:list|list-image.html]]      参照pro.html模板
  * [[ued:template:list|list-imagetext.html]]  也参照pro.html模板
  * [[ued:template:list|list-page.html]]       参照about.html模板
))

==== 表单嵌入模板规则 ====
1、需要添加form.json文件，文件格式如下：
{
    "bind_div0": {
        "value": [
        ],
        "description": "绑定显示区域",
        "type": "form",
        "config": {
            "kind": "div",
            "name": "div_area0,div_area1"
        }
    },
    "bind_css0": {
        "value": [
        ],
        "description": "绑定样式",
        "type": "form",
        "config": {
            "kind": "css",
            "name": "form_css0,form_css1"
        }
    },
    "bind_form0": {
        "value": [
        ],
        "description": "绑定表单",
        "type": "form",
        "config": {
            "kind": "form_id",
            "form_id": "1"
        }
    }
}

2、.html 文件

  <!--表单嵌入-->
  **<div id="div_area0">**
  <form method="post">
  <p class="kz" data-limit="58" //data-type="title"//></p>
  <p class="kz" data-limit="58" //data-type="description"//>{$index.word}</p>
  <input type="text" placeholder="学生姓名" class="short short1" //data-type="inset"// />
  <input type="text" placeholder="年龄" class="short" //data-type="inset"///>
  <input type="text" placeholder="联系电话" class="long" //data-type="inset"///>
  **<input type="submit" value="立即预约" name="submit" />**
  </form>
  </div>
  <!--表单嵌入end-->
                            
    ·表单嵌入的区域根据id来定位，id值要在json文件的bind_div中存在。
    ·data-type="title" 是表单标题
    ·data-type="description" 表单描述
    ·data-type="inset" 表单列，模板中的inset个数应该和嵌入的表单列个数相同
    ·每个表单都需要一个提交按钮，value值可以自定义
    
3、.css样式文件    

   ·表单嵌入的样式值要在json文件的bind_css中存在，并且对样式的设置要在该类名下。

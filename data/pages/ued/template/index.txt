  * ③[[#轮滚图定义方法|轮滚图定义]]

④⑤⑥⑦⑧⑨⑾使用到的也是[[#index_pc首页数据_自定义于indexjson文件中|index.json]]文件忠的一个类型为list的变量

⑾仅展示了某个栏目下的几篇（重要）文章，可选择使用[[#文章列表定义方法|文章列表]]中的输出__星标文章__模式：\\
只需要在[[#index_pc首页数据_自定义于indexjson文件中|index.json]]文件定义一个新的[[#typelist|list变量]]（假设list2）中设置''"star_only": 1''，即可将此列表默认值设置为__仅输出星标文章__！
  * ④⑤⑾[[#文章列表定义方法|文章列表定义]]
  * ⑥引用列表的[[#typelist|{$index.list1.data[]}]]变量说明（假设[[#index_pc首页数据_自定义于indexjson文件中|index.json]]中定义的变量名为''list1''）
  * ⑦引用列表的[[#typelist|{$index.list1.image}]]变量说明（假设[[#index_pc首页数据_自定义于indexjson文件中|index.json]]中定义的变量名为''list1''）
  * ⑧引用列表的[[#typelist|{$index.list1.name}]]变量说明（假设[[#index_pc首页数据_自定义于indexjson文件中|index.json]]中定义的变量名为''list1''）
  * ⑨引用列表的[[#typelist|{$index.list1.link}]]变量说明（假设[[#index_pc首页数据_自定义于indexjson文件中|index.json]]中定义的变量名为''list1''）

⑩⒀⒁中的LOGO不同于系统[[ued:template:header#logo_网站logo|{$logo}]]变量，因为[[ued:template:header#logo_网站logo|{$logo}]]已用于网站顶部大LOGO图，所以此处2号小LOGO可以在[[#index_pc首页数据_自定义于indexjson文件中|index.json]]文件中新定义个[[ued:template:json#typeimage|类型为image]]格式的变量，以引用
  * ⑩[[ued:template:json#typeimage|index.json中type:image的变量]]
  * ⒀[[ued:template:json#typeimage|{$index.qrcode.image}]]变量说明（假设[[ued:template:json|index.json]]中定义的变量名为''qrcode''）
  * ⒁[[ued:template:json#typeimage|{$index.qrcode.title}]]变量说明（假设[[ued:template:json|index.json]]中定义的变量名为''qrcode''）


⒂版权信息可放入[[ued:template:footer|变量语法查询：_footer.html]]文件中，利用[[ued:template:footer#include_引入文件|include]]方式引入到index.html文件中；

==== 留言板定义方法 ====

需在json文件中先定义一个[[ued:template:json#typenav|类型type为nav]]的变量，\\
然后在index.html文件中引用！

<file javascript index.json>
{
	"feedback": {
		"value": {},
		"description": "留言板",
		"type": "nav",
		"config": {
			"filter": "feedback"
		}
	}
}
</file>
